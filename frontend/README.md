# XAUUSD Mission Control - Frontend

The frontend interface mapped specifically connecting over to the FastAPI trading backend. Built using React + Next.js 14 and stylized fully with standard TailwindCSS constraints representing a premium interactive dashboard surface.

## Setup Instructions

1. **Install Dependencies:**
   Navigate into the `frontend` folder natively and run `npm install`:
   ```bash
   cd frontend
   npm install
   ```

2. **Launch Dev Environment:**
   Start the local Next deployment explicitly mapping over to browser via:
   ```bash
   npm run dev
   ```

3. **Verify UI:**
   Load [http://localhost:3000](http://localhost:3000) inside your web browser sequentially!
